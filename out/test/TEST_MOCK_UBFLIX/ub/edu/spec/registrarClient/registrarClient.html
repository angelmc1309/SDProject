UC 1: Requeriments per registrar-se:

REQ-01-00-01 Registre correcte:

Si totes les dades son vàlides i el client no existeix, es registrarà al client i creara l'usuari:
Username Password Resultat
ecolero ecolero Client afegit
atormenta atormenta Client afegit
username password Client afegit

REQ-01-00-02 Client ja registrat:

Si les dades del client son les d'un client ja registrat indicara al client el problema i permetra canviar les dades:
Username Password Resultat
ajaleo pass Usuari ja existeix
atormenta atormenta Usuari ja existeix
lmento password Usuari ja existeix
